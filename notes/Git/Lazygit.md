### Commit
Files panel -> `Space` to stage/unstage.
`c` -> commit message -> `Enter`

### Push
Commits panel -> `P`

### Branching

Branch panel
`n` = create new branch

### Merging

`f` = fetch from remote
Go to branch panel
Highlight master
`M` = merge master into your current branch
`e` = on a merge conflict to open it in your editor

### Switch branch

Branch panel -> select branch -> `Space`

### Navigating around panels

`0, 1, 2, 3, 4, 5` - D'OH!

### Seeing file history / commits by a person
Go to commit window -> Ctrl+S -> Search pattern -> Voila!

### Searching for patterns in different views
`/` , that's it, similar to vim

### Bisect
Press `b` in the commit window.
