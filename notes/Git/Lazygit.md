### Commit
Files panel -> `Space` to stage/unstage.
`c` -> commit message -> `Enter`

### Push
Commits panel -> `P`

### Branching
`b` = branches panel
`n` = create new branch

### Merging
`f` = fetch from remote
Go to branch panel
Highlight master
`M` = merge master into your current branch
`e` = on a merge conflict to open it in your editor
